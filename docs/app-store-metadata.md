### Notes for Reviewer
```
StatFocus is a fully local Pomodoro timer + stats app for macOS. No accounts, no network, no third-party services.

PERMISSIONS:
- File picker: only when user manually selects a backup folder or imports a backup
- Notifications: optional, asked only when user enables sound alerts
- Login items: optional, asked only when user enables "Start at login"

FOCUS BLOCKER FEATURE:
In this App Store build, the focus blocker only shows a friendly banner ("Focus active — you opened X") when the user switches to another app during a focus session. It does NOT force-hide or activate any other app — sandbox-compatible by design. Strict blocking and website blocking are only present in the direct-distribution Developer ID build, not here.

To test: open the app, go to Stats / Settings, configure a focus session, switch to another app — you'll see the floating banner.
```
